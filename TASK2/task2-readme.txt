อธิบายฟังก์ชัน
print_json(data)
ฟังก์ชันรับข้อมูลมาจาก data ที่เป็นข้อมูลที่ได้จาก API ฟังก์ชันจะทำการ print ข้อมูลที่ได้มาจาก data ที่เก็บใน key แล้วแสดงออกทาง terminal รายการที่ดึงค่าของแต่ละรายการด้วย data.get(key)

save_poster(data)
ฟังก์ชันรับข้อมูลมาจาก data ที่เป็นข้อมูลที่ได้จาก API ฟังก์ชันจะทำการตรวจสอบว่ามีลิ้งค์ของโปรเตอร์หรือไม่ ถ้ามีจะทำการสร้างไฟล์ด้วยรอูแบบ ชื่อภาพยนต์_poster_6609612061.jpg และใช้ urllib.requst.urlopen() เพื่อดาวน์โหลดรูปโปรเตอร์ แล้วจะเปิดไฟล์แล้วเขียนข้อมูลของภาพลงไป

search_movie(title)
ฟังก์ชันรับข้อมูลมาจาก title ที่รับมาจาก terminal แล้วจะแปลง title ให้ใช้ได้ใน URL ฟังก์ชันจะประกอบ URL ที่ใช้เรียก API โดยใช้ API key แล้วใช้ urllib.request.urlopen() เพื่อส่ง request แล้วจะอ่านผลลัพธ์ JSON แล้วแปลงเป็น python dict ด้วย json.load() 

มอดูลที่ใช้
- urllib.request	ใช้ดึงข้อมูล JSON และ ดาวน์โหลดโปสเตอร์
- urllib.parse	ใช้แปลงชื่อหนังเป็น URL
- urllib.error	ใช้ตรวจสอบข้อมูลผิดพลาดจากการโหลด URL
- json			ใช้แปลง JSON ให้เป็น python oject
